{{< /blocks/cover >}} {{% blocks/lead color="white" %}} **Kuesta** is an open source framework for network configuration, enabling
Infrastructure-as-Code of network devices. It provides us a rich network automation
by adopting cloud-native best practices of IaC and GitOps. {{% /blocks/lead %}} {{< blocks/section color="primary" >}} {{% blocks/feature icon="fa-feather" title="Usecase Agnostic" %}} No constraints for networking model,
just focusing on data mapping, validation, and provisioning.
Write your own network service just you want. {{% /blocks/feature %}} {{% blocks/feature icon="fa-file-code" title="Programmability" %}} Using **CUE** for data mapping provides
data validation, code generation, and software practices
suited for configuration. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="GitOps" %}} Pull-based approach leveraging Kubernetes Custom Operator ensures **Single Source of Truth** of configuration repository and secure git-based operation. {{% /blocks/feature %}} {{< /blocks/section >}}
